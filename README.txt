							Seja Bem vindo ao meu projeto!!!

#Sobre:

- Este projeto foi desenvolvido como parte de um desafio oferecido pela parceria entre Alura e Oracle Next Education (Projeto ONE). 
O objetivo é complementar o aprendizado colocando em prática os conhecimentos adquiridos.

- O desafio é baseado no sorteio de um amigo secreto.


#Desenvolvido por: Ian Oliveira de Almeida

#Instruções:

1. - Digite o nome de um dos amigos no campo "Digite um nome". (Caso você insira apenas um nome, o código só retornará o único nome).
2. - Após inserir um nome, clique no botão "Adicionar". (Se o nome foi inserido corretamente, ele aparecerá numa lista abaixo do campo "Digite um nome").
3. - Repita esse processo para cada pessoa.
4. - Após adicionar todos os nomes, clique no botão laranja "Sortear amigo". O resultado exibirá o seu amigo secreto.

#Possíveis Problemas:

- **Nenhum nome digitado**: O sistema alertará o usuário caso não tenha sido digitado um nome no campo.
- Solução: Verifique se está realmente digitando o nome no campo correto seguindo as Instruções logo acima.

- **Nome duplicado**: O sistema alertará o usuário caso um nome duplicado seja inserido.
- Solução: Busque inserir o sobre nome ou o apelido, pois se houver 2 pessoas com o mesmo nome não saberá diferenciar os nomes.

- **Nenhum nome inserido**: O sorteio não será permitido se a lista estiver vazia e o usuário será alertado.
- O sorteio só é possível caso tenha nomes já adicionados no jogo.
